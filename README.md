# 🚀 Recursion in C++

A comprehensive collection of **Recursion programs in C++** designed to strengthen problem-solving skills and build a solid foundation in **Data Structures & Algorithms (DSA)**.

This repository contains beginner to intermediate recursion problems, along with searching and sorting algorithms implemented using clean and readable C++ code.

---

## 📌 Topics Covered

### 🔹 Basic Recursion
- Print 1 to N
- Sum of N Numbers
- Sum of Squares
- Factorial
- Power of a Number
- Reverse String
- Check Palindrome
- Sum of Array
- Print Array
- Minimum Element in Array

### 🔹 Searching Algorithms
- Linear Search
- Binary Search

### 🔹 Sorting Algorithms
- Merge Sort
- Quick Sort

### 🔹 String Problems
- Count Vowels
- Lowercase to Uppercase Conversion

### 🔹 Array Problems
- Sum of Array
- Minimum Element
- Print Array

### 🔹 Miscellaneous
- Even Number Check
- Practice Recursion Problems (Rec1, Rec2)
